*** Flask:
Python Version 3.5 and above.

Optional dependencies
	These distributions will not be installed automatically. Flask will detect and use them if you install them.
	1. Blinker: It provides support for Signals.
	2. SimpleJSON: It, is a fast JSON implementation that is compatible with Python’s json module. It is preferred for JSON operations if it is installed.
	3. python-dotenv: It, enables support for Environment Variables From dotenv when running flask commands.
	4. Watchdog: It, provides a faster, more efficient reloader for the development server
	
***Virtual environments:
 -Use a virtual environment to manage the dependencies for your project, both in development and in production.
 -Virtual environments are independent groups of Python libraries, one for each project. Packages installed for one project will not affect other projects or the operating system’s packages. 
 -Python 3 comes bundled with the venv module to create virtual environments.

	After install the below dependencies comes by default:
		These distributions will be installed automatically when installing Flask.
		1. Werkzeug: It, implements WSGI, the standard Python interface between applications and servers.
		2. Jinja: It is a template language that renders the pages your application serves.
		3. MarkupSafe: It comes with Jinja. It escapes untrusted input when rendering templates to avoid injection attacks.
		4. ItsDangerous: It securely signs data to ensure its integrity. This is used to protect Flask’s session cookie.
		5. Click: It is a framework for writing command line applications. It provides the flask command and allows adding custom management commands.
